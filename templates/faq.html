Washboard

FAQ

  • What is Washboard?

    Washboard is a web app that provides an alternative frontend to Tumblr. Its primary goal is to give mobile users a way to blacklist keywords, since mobile devices don't have Tumblr Savior or XKit.

  • How does signing up work?

    Washboard guides the user through the steps to register their own Tumblr application and then authenticates the user with their own app. The user's primary blog name is used for their Washboard username.

  • What happens if I change my username on Tumblr?

    After changing your Tumblr username, you can log in once using your old username. After doing so, Washboard will update your username automatically and inform you of the change.

  • How does blacklisting work?

    The "Rules" page allows users to manage their blacklisted and whitelisted keywords. Words or phrases can be blacklisted, and there are per-keyword options to show a notification upon blacklisting and to only match the whole world (as opposed to the keyword "board" matching "Washboard", for example).

  • Will the blacklist stop working if Tumblr updates the layout?

    Nope. Although this happens occasionally with desktop browser extensions, Washboard uses the Tumblr API, which is relatively stable. In the event that something critical breaks, Washboard will not display posts at all, rather than displaying all posts indiscriminately.

  • It's not working! What do I do?

    Please email me or send me an ask with as many details as you can provide. Your feedback is critical!

  • Does Washboard comply with Tumblr's API License Agreement?

    I believe so, although there are a few grey areas.

    • Because the API keys are generated by each user, the "Obtaining and Maintaining an API Key" clause suggests that uploading the keys to Washboard is prohibited. However, the keys are stored securely and serve much the same purpose as OAuth access tokens, which have to be stored by any application that uses the Tumblr API.
    • The "No Substantial Replication" clause prohibits full recreation of the Tumblr Dashboard, but permits "specialized views". Since Washboard provides extra functionality to the Dashboard, it is compliant with this clause.
    • The "No Modification of Tumblr Dashboards or Content Streams" clause prohibits removal of content from the Dashboard. Blacklisted posts are hidden by Washboard, but only because the user explicitly requested for them to be hidden. (As an aside: if in-stream advertisements are added to the API in the future, they will always be displayed to the user.)
    • All other clauses of the API License Agreement are unambiguously fulfilled by Washboard.
